  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Debentialc up and running on your Minecraft server in minutes
  </p>

  <Steps>
    <Step title="Download and Install">
      Download the latest Debentialc JAR file and place it in your server's `plugins` folder. Make sure you're running Bukkit or Spigot 1.8 or higher.

      ```bash
      # Place the plugin JAR in your plugins folder
      cp Debentialc-1.1.0.jar /path/to/server/plugins/
      ```
    </Step>

    <Step title="Install Dependencies">
      Debentialc requires several optional plugins for full functionality:
      
      - **PlaceholderAPI** - For placeholder support in messages and scoreboards
      - **Vault** - For economy integration with territory claims
      - **CustomNPCs** - For the raid system to work with NPCs

      ```bash
      # Download and install dependencies in your plugins folder
      # These are optional but recommended for full features
      ```

      <Note>
        The plugin will work without these dependencies, but some features may be disabled. Check the console on startup for dependency status.
      </Note>
    </Step>

    <Step title="Start Your Server">
      Start or restart your Minecraft server. Debentialc will automatically initialize all systems and create default configuration files.

      ```bash
      # Check the console for successful initialization
      [Debentialc] Plugin successfully enabled
      [Debentialc] Version: 1.1.5
      [Boosters] Sistema de Boosters inicializado correctamente
      [Raids] Sistema de raids inicializado
      [Claims] Sistema de terrenos inicializado
      ```
    </Step>

    <Step title="Configure and Test">
      Use the in-game commands to start using Debentialc features. Try activating your first booster or creating a custom item:

      ```bash
      # In-game commands to test
      /booster help
      /customitem help
      /raid help
      /terrain help
      ```

      <Accordion title="Example: Activate a Global Booster">
        ```bash
        /booster activate 2.0
        ```
        This activates a 2.0x multiplier for all players on the server for the configured duration.
      </Accordion>
    </Step>
  </Steps>
</div>
