The booster system provides temporary multipliers that enhance player rewards and experience. There are two types of boosters: global boosters that affect all players on the server, and personal boosters that only affect individual players.

## Usage Examples

### Creating and Activating a Global Booster

```java
import org.debentialc.boosters.models.GlobalBooster;
import java.time.Instant;

// Create a 50% bonus booster for 1 hour
Instant start = Instant.now();
Instant end = start.plusSeconds(3600);
GlobalBooster booster = new GlobalBooster(1.5, start, end, "ServerAdmin");

// Check status
if (booster.isActive() && !booster.hasExpired()) {
    System.out.println("Booster active: " + booster.getPercentageBonus() + "% bonus");
    System.out.println("Time remaining: " + booster.getFormattedTime());
}
```

### Managing Personal Boosters

```java
import org.debentialc.boosters.models.PersonalBooster;
import java.util.UUID;

UUID playerId = player.getUniqueId();

// Create a level 3 booster with 75% bonus
PersonalBooster booster = new PersonalBooster(playerId, 3, 1.75);

// Activate for 30 minutes
long duration = 1800; // 30 minutes in seconds
booster.activate(duration);

// Check if still active
if (booster.isStillActive(duration)) {
    long remaining = booster.getActivationTimeRemaining(duration);
    player.sendMessage("Your " + booster.getLevelName() + " booster has " + remaining + " seconds remaining!");
}
```

### Checking Storage Expiration

```java
long maxStorageDays = 30;

if (booster.hasExpiredFromStorage(maxStorageDays)) {
    System.out.println("Booster has expired from storage");
} else {
    long remainingSeconds = booster.getStorageTimeRemaining(maxStorageDays);
    long remainingDays = remainingSeconds / 86400;
    System.out.println("Booster expires in " + remainingDays + " days");
}
```
