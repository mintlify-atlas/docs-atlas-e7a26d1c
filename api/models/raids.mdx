The raid system provides cooperative PvE challenges where groups of players fight through waves of enemies. The system includes multiple interconnected models that manage raid configurations, active sessions, player parties, and wave progression.

## Usage Examples

### Creating and Configuring a Raid

```java
import org.debentialc.raids.models.*;
import org.bukkit.Location;

// Create a new raid
Raid raid = new Raid("dragon_lair", "Dragon's Lair");
raid.setDescription("Face the ancient dragon and its minions");
raid.setPlayerSpawnPoint(playerSpawn);
raid.setMinPlayers(2);
raid.setMaxPlayers(4);
raid.setCooldownSeconds(7200); // 2 hours

// Add waves
for (int i = 1; i <= 5; i++) {
    Wave wave = new Wave(i);
    wave.setDescription("Wave " + i + " - Increasing difficulty");
    raid.addWave(wave);
}

if (raid.isConfigured()) {
    System.out.println("Raid is ready with " + raid.getTotalWaves() + " waves");
}
```

### Managing a Raid Session

```java
import java.util.UUID;

// Create a party
UUID leaderUuid = player.getUniqueId();
Party party = new Party("party_123", leaderUuid, 4);

// Start a raid session
RaidSession session = new RaidSession("session_456", raid, party);

System.out.println("Starting raid: " + session.getRaid().getRaidName());
System.out.println("Wave 1 of " + session.getRaid().getTotalWaves());

// Track progress
while (session.isWaveActive() && !session.isRaidFailed()) {
    int progress = session.getProgress();
    Wave currentWave = session.getCurrentWave();
    
    System.out.println("Progress: " + progress + "% - Wave " + 
                       (session.getCurrentWaveIndex() + 1));
    
    // ... wave completion logic ...
    
    if (session.hasNextWave()) {
        session.moveToNextWave();
    } else {
        break;
    }
}

long duration = session.getDurationSeconds();
System.out.println("Raid completed in " + duration + " seconds");
```

### Handling Player Events

```java
// Player dies
UUID playerId = player.getUniqueId();
session.playerDied(playerId);

if (session.isRaidFailed()) {
    System.out.println("All players defeated! Raid failed.");
}

// Player leaves
session.playerLeft(playerId);

// Check if player can return
if (session.canPlayerRejoin(playerId)) {
    session.playerReturned(playerId);
    System.out.println("Player rejoined the raid!");
}
```

### Working with Parties

```java
Party party = new Party("party_789", leaderUuid, 5);

// Add members
for (Player member : partyMembers) {
    if (!party.isFull()) {
        party.addMember(member.getUniqueId(), false);
    }
}

System.out.println("Party size: " + party.getMemberCount() + "/" + party.getMaxSize());

if (party.canStartRaid()) {
    if (party.isLeader(player.getUniqueId())) {
        // Start raid
        System.out.println("Leader starting raid...");
    }
}

// Get party age
long age = party.getAgeSeconds();
System.out.println("Party has existed for " + age + " seconds");
```
