The `CustomItem` class is the core model for the custom items system, allowing you to create items with custom attributes, stats, effects, durability, and scripted behaviors.

## Package

```java
org.debentialc.customitems.tools.ci.CustomItem
```

## Overview

CustomItem provides a builder-pattern API for creating items with:
- Custom display names and lore
- Stat bonuses and operations
- Potion effects
- Custom durability
- Consumable properties
- Command execution on use
- TP (Training Points) values
- Attack damage modifiers
- NBT data

Sets the material type for the item.

```java

```

**Parameters:**
- `material` (int): The Minecraft material ID

**Returns:** `CustomItem` - The current instance for chaining

**Example:**

## Usage Examples

### Creating a Basic Custom Item

```java
import org.debentialc.customitems.tools.ci.CustomItem;
import java.util.Arrays;

CustomItem sword = new CustomItem()
    .setId("legendary_sword")
    .setMaterial(276) // Diamond Sword
    .setDisplayName("§6§lLegendary Blade")
    .setLore(Arrays.asList(
        "§7A powerful weapon forged",
        "§7in the fires of Mount Doom",
        "",
        "§c+15 Strength",
        "§9+10 Critical Damage"
    ));
```

### Adding Stats and Effects

```java
CustomItem armor = new CustomItem()
    .setId("guardian_chestplate")
    .setMaterial(311) // Diamond Chestplate
    .setDisplayName("§b§lGuardian's Protection")
    .setBonusStat("defense", 20.0)
    .setBonusStat("health", 15.0)
    .setMaxDurability(1000)
    .setUnbreakable(false);

// Add regeneration effect
HashMap<String, Double> effects = new HashMap<>();
effects.put("REGENERATION", 0.0);
effects.put("RESISTANCE", 0.0);
armor.setEffects(effects);
```

### Creating a Consumable Item

```java
import java.util.Arrays;

CustomItem potion = new CustomItem()
    .setId("healing_elixir")
    .setMaterial(373) // Potion
    .setDisplayName("§d§lHealing Elixir")
    .setConsumable(true)
    .setCommands(Arrays.asList(
        "effect @dp 10 10 5", // Regeneration
        "effect @dp 11 2 5"   // Resistance
    ));
```

### Creating a TP Reward Item

```java
CustomItem tpScroll = new CustomItem()
    .setId("tp_scroll")
    .setMaterial(339) // Paper
    .setDisplayName("§e§lTraining Scroll")
    .setLore(Arrays.asList(
        "§7Use to gain 1000 TP",
        "§7Consumes the entire stack"
    ))
    .setTpValue(1000)
    .setTpConsumeStack(true)
    .setConsumable(true);
```

### Creating an Item with Multiple Stats

```java
HashMap<String, Double> stats = new HashMap<>();
stats.put("strength", 25.0);
stats.put("defense", 15.0);
stats.put("speed", 10.0);
stats.put("criticalDamage", 30.0);

HashMap<String, String> operations = new HashMap<>();
operations.put("strength", "ADD");
operations.put("defense", "ADD");
operations.put("speed", "MULTIPLY");
operations.put("criticalDamage", "ADD");

CustomItem epicSword = new CustomItem()
    .setId("epic_blade")
    .setMaterial(276)
    .setDisplayName("§5§lEpic Blade of Legends")
    .setValueByStat(stats)
    .setOperation(operations)
    .setAttackDamage(15)
    .setMaxDurability(2000);
```
