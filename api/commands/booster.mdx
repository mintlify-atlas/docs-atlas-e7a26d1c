# Booster Commands

The booster system provides temporary multipliers to enhance player rewards. Boosters can be global (server-wide) or personal (individual players).

## Base Commands

### /booster

**Aliases:** `boosters`, `boost`

**Permission:** `debentialc.command.booster`

**Usage:** `/booster <subcommand>`

Main command for managing the booster system. Without arguments, displays the help menu.

---

## Global Booster Subcommands

Global boosters affect all players on the server simultaneously.

### /booster global activate

**Usage:** `/booster global activate <percentage> [time]`

Activates a server-wide booster with the specified percentage bonus.

<ParamField path="percentage" type="string" required>
  The bonus percentage (e.g., `50%`, `150%`, `200%`). Must include the `%` symbol.
</ParamField>

<ParamField path="time" type="string">
  Duration in time format (e.g., `1h`, `2h30m`, `1d`). If omitted, uses the default duration from config.
  
  **Time format:**
  - `d` = days
  - `h` = hours
  - `m` = minutes
  - `s` = seconds
</ParamField>

**Examples:**

```bash
# Activate 50% bonus with default duration
/booster global activate 50%

# Activate 150% bonus for 1 hour
/booster global activate 150% 1h

# Activate 200% bonus for 2 hours 30 minutes
/booster global activate 200% 2h30m
```

**Output:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚡ BOOSTER GLOBAL ACTIVADO ⚡

  Bono: +150%
  Duración: 1h
  Activado por: PlayerName

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

<Note>
  When activated, all players receive a broadcast message announcing the global booster.
</Note>

---

### /booster global deactivate

**Usage:** `/booster global deactivate`

Immediately deactivates the current global booster.

**Example:**

```bash
/booster global deactivate
```

**Output:**

```
Booster global desactivado
[BOOSTER] Booster Global Desactivado (broadcast)
```

---

### /booster global info

**Aliases:** `multiplier`

**Usage:** `/booster global info`

Displays information about the currently active global booster.

**Example:**

/booster global info
```

**Output:**

```

Bono: +150%

<Warning>

</Warning>

## Personal Booster Subcommands

Personal boosters are individual player bonuses that can be activated independently.

### /booster personal add

**Usage:** `/booster personal add <player> <level>`

Gives a personal booster to a player.

<ParamField path="player" type="string" required>
  Target player's username (must be online)
</ParamField>

<ParamField path="level" type="integer" required>
  Booster level (1-5). Higher levels provide greater multipliers.
</ParamField>

**Example:**

```bash
# Give level 3 booster to player
/booster personal add Steve 3
```

**Output (sender):**

```
Booster nivel 3 añadido a Steve
  Bonus: +75%
```

**Output (recipient):**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚡ BOOSTER RECIBIDO ⚡

  Nivel: 3
  Bono: +75%

Usa /booster personal list para ver tus boosters
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

<Note>
  Booster multipliers are configured in `BoosterSettings`. Each level (1-5) has a predefined multiplier.
</Note>

---

### /booster personal activate

**Usage:** `/booster personal activate <index>`

Activates one of your personal boosters by index.

<ParamField path="index" type="integer" required>
  The index of the booster to activate (0-based). Use `/booster personal list` to see available boosters and their indices.
</ParamField>

**Example:**

```bash
# Activate the first booster (index 0)
/booster personal activate 0
```

**Output:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚡ BOOSTER ACTIVADO ⚡

  Nivel: Level III
  Bono: +75%
  Duración: 30m

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

---

### /booster personal list

**Usage:** `/booster personal list`

Lists all your available personal boosters.

**Example:**

```bash
/booster personal list
```

**Output:**

```
=== Tus Boosters Personales ===
  [0] Level I - +25% (Disponible)
  [1] Level III - +75% (Activo)
  [2] Level V - +150% (Disponible)

Usa /booster personal activate <índice> para activar
```

## Data Management

### /booster save

**Usage:** `/booster save`

Manually saves all booster data to disk.

**Example:**

```bash
/booster save
```

**Output:**

```
Datos guardados exitosamente
```

<Note>
  Booster data is automatically saved periodically. This command is for manual backups.
</Note>

---

### /booster load

**Usage:** `/booster load`

Reloads booster data from disk.

**Example:**

```bash
/booster load
```

**Output:**

```
Datos cargados exitosamente
```

<Warning>
  This will overwrite current in-memory booster data with saved data.
</Warning>

## TP Booster Commands

Special commands for applying boosters to TP (Training Points) rewards.

### /tpbooster give

**Aliases:** `tpboost`

**Permission:** `debentialc.command.tpbooster`

**Usage:** `/tpbooster give <amount>`

Gives the player TPs with active booster multipliers applied.

<ParamField path="amount" type="integer" required>
  Base amount of TPs to give (before booster multiplication)
</ParamField>

**Example:**

```bash
# Give 100 TPs with booster applied
/tpbooster give 100
```

---

### /tpbooster info

**Usage:** `/tpbooster info`

Displays your current TP booster status and multiplier.

**Example:**

```bash
/tpbooster info
```

**Output:**

```
=== Info de Boosters de TPs ===

  Estado: Activo
  Multiplicador: x2.50
  Bonus: +150%
```

---

### /tpbooster test

**Usage:** `/tpbooster test`

Shows example calculations with your current booster.

**Example:**

```bash
/tpbooster test
```

**Output:**

```
=== Test de Boosters de TPs ===

  10 TPs → 25 TPs (+15) [x2.50]
  50 TPs → 125 TPs (+75) [x2.50]
  100 TPs → 250 TPs (+150) [x2.50]
  500 TPs → 1250 TPs (+750) [x2.50]
  1000 TPs → 2500 TPs (+1500) [x2.50]
```

---

### /tpbooster calculate

**Usage:** `/tpbooster calculate <amount>`

Calculates how many TPs you would receive with your current booster.

<ParamField path="amount" type="integer" required>
  Base TP amount to calculate
</ParamField>

**Example:**

```bash
/tpbooster calculate 250
```

**Output:**

```
=== Cálculo de Boosters ===

  TPs Base: 250
  Bonus: +375
  Total: 625 TPs
  Multiplicador: x2.50
```

---

### /dartps

**Permission:** `debentialc.dartps`

**Usage:** `/dartps <player> <amount>`

Admin/console command to give TPs to a player with booster applied.

<ParamField path="player" type="string" required>
  Target player username
</ParamField>

<ParamField path="amount" type="integer" required>
  Base TP amount (booster will be applied)
</ParamField>

**Example:**

```bash
/dartps Steve 500
```

<Note>
  This command can be executed from console and will apply the player's active boosters.
</Note>

## Implementation Details

### Source Files

- **BoosterCommand.java** - Line 23: Main booster command handler

### Multiplier Calculation

Boosters stack multiplicatively:

```java
// Global booster: x2.00 (+100%)
// Personal booster: x1.50 (+50%)
// Combined: 2.00 * 1.50 = x3.00 total multiplier
```

### Time Format Parsing

Supported time units:
- `d` = days
- `h` = hours  
- `m` = minutes
- `s` = seconds

Examples: `1d`, `2h30m`, `45s`, `1d12h30m`
