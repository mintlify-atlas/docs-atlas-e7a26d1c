## Overview

The Custom Items System provides a powerful framework for creating items with custom stat bonuses, effects, and enhancements for Dragon Block C. It includes support for **five core stats** (STR, CON, DEX, WILL, MND), **regeneration effects**, **armor fragments**, and **tiered customization** with operation-based bonuses (+, -, ×).

## Core Stats System

### Available Stats

The system supports five primary stats that integrate with Dragon Block C:

| Stat | DBC Attribute | Description |
|------|--------------|-------------|
| **STR** | Strength | Increases melee damage and ki damage |
| **CON** | Constitution | Increases health points and defense |
| **DEX** | Dexterity | Increases speed and evasion |
| **WILL** | Willpower | Increases ki pool and ki regeneration |
| **MND** | Mind | Increases special attack damage |

### Stat Operations

Each stat bonus uses one of three operations:

<CardGroup cols={3}>
  <Card title="Addition (+)" icon="plus">
    Adds a flat value to the stat
    
    Example: `+500 STR`
  </Card>
  <Card title="Subtraction (-)" icon="minus">
    Subtracts a value from the stat
    
    Example: `-200 DEX`
  </Card>
  <Card title="Multiplication (×)" icon="xmark">
    Multiplies the stat by a percentage
    
    Example: `×15%` = 1.15x multiplier
  </Card>
</CardGroup>

<Note>
**Percentage Multipliers:** When using `*` operation with percentages:
- `15%` internally becomes `1.15` (add 15% to base)
- `-20%` internally becomes `0.80` (reduce by 20%)
- `100%` internally becomes `2.0` (double the value)
</Note>

### Value Parsing System

**Location:** `customitems/tools/fragments/ArmorFragment.java:53`

```java
private void parseValue(String value) {
    if (value.endsWith("%")) {
        // PERCENTAGE: "15%" → operation "*", value 1.15
        this.operation = "*";
        String numStr = value.substring(0, value.length() - 1);
        double percentage = Double.parseDouble(numStr);
        this.numericValue = 1.0 + (percentage / 100.0);
    } else if (value.startsWith("-")) {
        // NEGATIVE: "-500" → operation "-", value 500
        this.operation = "-";
        this.numericValue = Math.abs(Double.parseDouble(value));
    } else {
        // POSITIVE: "500" → operation "+", value 500
        this.operation = "+";
        this.numericValue = Double.parseDouble(value);
    }
}
```

<Warning>
**Percentage Conversion:**
- User input: `"15%"`
- Stored as: `1.15` (DBC multiplier)
- Displayed as: `"15%"` (user-friendly)

Internal values are **always** DBC-compatible multipliers.
</Warning>

### Fragment Application

**Location:** `customitems/tools/fragments/FragmentManager.java:36`

```java
public boolean applyFragment(Player player, ItemStack fragment, ItemStack armor) {
    // 1. Validate fragment
    if (!ArmorFragment.isFragment(fragment)) {
        player.sendMessage("§c✗ Este no es un fragmento válido");
        return false;
    }
    
    // 2. Extract fragment data
    String attribute = ArmorFragment.getFragmentAttribute(fragment);
    String operation = ArmorFragment.getFragmentOperation(fragment);
    double value = ArmorFragment.getFragmentValue(fragment);
    
    // 3. Load or create customized armor
    CustomizedArmor customArmor;
    if (CustomizedArmor.isCustomized(armor)) {
        customArmor = CustomizedArmor.fromItemStack(armor);
    } else {
        customArmor = convertVanillaArmor(armor);
    }
    
    // 4. Validate tier limits
    if (!tierConfig.canApply(customArmor.getTier(), attribute, 
                              currentValue, valueToAdd, operation)) {
        player.sendMessage("§c✗ El valor excedería el límite");
        return false;
    }
    
    // 5. Calculate new value based on operation
    int currentValue = customArmor.getAttributeValue(attribute);
    int newValue;
    
    switch (operation) {
        case "+":
            newValue = currentValue + (int) value;
            break;
        case "-":
            newValue = currentValue - (int) value;
            break;
        case "*":
            // Sum PERCENTAGES for multiplicative bonuses
            int scaledFragmentValue = (int) Math.round(value * 100);
            String currentOp = customArmor.getOperations().get(attribute);
            
            if (currentOp == null || !currentOp.equals("*")) {
                // First multiplicative fragment
                newValue = scaledFragmentValue;
            } else {
                // Add percentages: 20% + 10% = 30%
                double currentPercentage = (currentValue / 100.0 - 1.0) * 100.0;
                double fragmentPercentage = (scaledFragmentValue / 100.0 - 1.0) * 100.0;
                double totalPercentage = currentPercentage + fragmentPercentage;
                newValue = (int) Math.round((1.0 + totalPercentage / 100.0) * 100);
            }
            break;
    }
    
    // 6. Apply and save
    customArmor.getAttributes().put(attribute, newValue);
    customArmor.getOperations().put(attribute, operation);
    customArmor.applyToItemStack(armor);
    armorStorage.saveArmor(customArmor);
    
    // 7. Consume fragment
    if (fragment.getAmount() > 1) {
        fragment.setAmount(fragment.getAmount() - 1);
    } else {
        player.setItemInHand(new ItemStack(Material.AIR));
    }
    
    return true;
}
```

<Tip>
**Tier Progression:**
- **Basic:** Only flat bonuses (+), limited values
- **Advanced:** Flat bonuses and penalties (+/-), higher limits
- **Elite:** All operations including multipliers (+/-/×), highest limits
</Tip>

### Fragment Tag System

**Location:** `customitems/tools/fragments/ArmorFragment.java:34`

Fragments are identified by a hidden NBT tag in the lore:

```java
private static final String FRAGMENT_TAG = "§8[FRAGMENT:%s:%s:%s]"; // attr:op:value

// Example tag in item lore (invisible to players):
// §8[FRAGMENT:STR:+:500]
// §8[FRAGMENT:DEX:*:15%]
```

**Detection Methods:**

```java
// Check if item is a fragment
public static boolean isFragment(ItemStack item) {
    if (item == null || !item.hasItemMeta()) return false;
    List<String> lore = item.getItemMeta().getLore();
    for (String line : lore) {
        if (line.contains("[FRAGMENT:")) return true;
    }
    return false;
}

// Extract attribute (STR, CON, etc.)
public static String getFragmentAttribute(ItemStack item) {
    String clean = extractFragmentTag(item);
    String[] parts = clean.split(":");
    return parts[0]; // "STR"
}

// Extract operation (+, -, *)
public static String getFragmentOperation(ItemStack item) {
    String clean = extractFragmentTag(item);
    String[] parts = clean.split(":");
    return parts[1]; // "+"
}

// Extract raw value ("500", "15%")
public static String getFragmentValueRaw(ItemStack item) {
    String clean = extractFragmentTag(item);
    String[] parts = clean.split(":");
    return parts[2]; // "15%"
}
```

## Custom Armor

### CustomizedArmor Class

**Location:** `customitems/tools/fragments/CustomizedArmor.java`

```java
public class CustomizedArmor {
    private String hash;                            // Unique identifier
    private String tier;                            // BASIC/ADVANCED/ELITE
    private int materialType;                       // Minecraft material ID
    private String armorSlot;                       // HELMET/CHESTPLATE/LEGGINGS/BOOTS
    private String displayName;
    private Map<String, Integer> attributes;        // Attribute → Value
    private Map<String, String> operations;         // Attribute → Operation
    
    // Apply attributes to ItemStack
    public void applyToItemStack(ItemStack item) {
        ItemMeta meta = item.getItemMeta();
        
        // Update lore with stats
        List<String> lore = new ArrayList<>();
        lore.add("§8[Tier: §7" + tier + "§8]");
        
        for (Map.Entry<String, Integer> entry : attributes.entrySet()) {
            String attr = entry.getKey();
            int value = entry.getValue();
            String op = operations.get(attr);
            
            if (op.equals("*")) {
                // Display as percentage
                double percentage = (value / 100.0 - 1.0) * 100.0;
                lore.add(String.format("§e%s: §a%+.0f%%", attr, percentage));
            } else {
                // Display as flat value
                lore.add(String.format("§e%s: §f%s%d", attr, op, value));
            }
        }
        
        // Add hash tag (hidden)
        lore.add("§8[ARMOR:" + hash + "]");
        
        meta.setLore(lore);
        item.setItemMeta(meta);
    }
}
```

public static String generateUniqueHash(Set<String> existingHashes) {
    String hash;
    do {
        hash = UUID.randomUUID().toString().substring(0, 8).toUpperCase();

public class EffectsManager {
    
    /**
     * Apply regeneration effects from equipped items
     */
    public static void applyEffects(Player player, CustomItem item) {
        if (item.getEffects() == null || item.getEffects().isEmpty()) {
            return;
        }
        
        IDBCPlayer dbcPlayer = getDBCPlayer(player);
        if (dbcPlayer == null) return;
        
        for (Map.Entry<String, Double> effect : item.getEffects().entrySet()) {
            String type = effect.getKey();
            double value = effect.getValue();
            
            switch (type) {
                case "HEALTHREGEN":
                    int maxHP = dbcPlayer.getMaxHealth();
                    int regenHP = (int) (maxHP * value);
                    dbcPlayer.setHealth(dbcPlayer.getHealth() + regenHP);
                    break;
                    
                case "KIREGEN":
                    int maxKi = dbcPlayer.getMaxKi();
                    int regenKi = (int) (maxKi * value);
                    dbcPlayer.setKi(dbcPlayer.getKi() + regenKi);
                    break;
                    
                case "STAMINAREGEN":
                    int maxStamina = dbcPlayer.getMaxStamina();
                    int regenStamina = (int) (maxStamina * value);
                    dbcPlayer.setStamina(dbcPlayer.getStamina() + regenStamina);
                    break;
            }
        }
    }
}
```

public class CustomItemStorage {
    private static final String BASE_PATH = "plugins/Debentialc/data/customitems";
    
    public void saveItem(CustomItem item) {
        Path filePath = Paths.get(BASE_PATH, "items", item.getId() + ".json");
        String json = GSON.toJson(item);
        Files.write(filePath, json.getBytes(StandardCharsets.UTF_8));
    }
    
    public Map<String, CustomItem> loadAllItems() {
        Map<String, CustomItem> items = new HashMap<>();
        Path itemsDir = Paths.get(BASE_PATH, "items");
        
        Files.walk(itemsDir)
            .filter(p -> p.toString().endsWith(".json"))
            .forEach(path -> {
                String json = new String(Files.readAllBytes(path));
                CustomItem item = GSON.fromJson(json, CustomItem.class);
                items.put(item.getId(), item);
            });
        
        return items;
    }
}
```

## Troubleshooting

<Accordion title="Stats not applying to player">
  **Solutions:**
  1. Verify item is registered: `/ci list`
  2. Check item in hand: `/ci info`
  3. Ensure DBC is loaded and player has DBC data
  4. Check console for calculation errors
</Accordion>

<Accordion title="Fragment application fails">
  **Solutions:**
  1. Check if armor is at tier limit
  2. Verify operation is allowed for tier
  3. Check fragment is valid: look for `[FRAGMENT:` in lore
  4. Try converting armor first with a basic fragment
</Accordion>

<Accordion title="Custom armor lost after relog">
  **Solutions:**
  1. Verify hash exists in `plugins/Debentialc/data/customitems/armor/`
  2. Check file permissions on data directory
  3. Look for JSON save errors in console
  4. Ensure armor has the `[ARMOR:hash]` tag in lore
</Accordion>
