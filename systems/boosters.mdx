## Overview

The Boosters System provides a comprehensive multiplier framework for TP (Training Points) rewards in Dragon Block C. It supports both **global boosters** (server-wide) and **personal boosters** (individual players), with configurable rank-based multipliers, persistent storage, and full PlaceholderAPI integration.

  </Card>

### GlobalBoosterManager

**Location:** `boosters/managers/GlobalBoosterManager.java:7`

Manages the single active global booster instance.

```java
public static void activateBooster(double multiplier, String activatedBy) {
    Instant now = Instant.now();
    Instant end = now.plusSeconds(durationSeconds);
    GlobalBooster booster = new GlobalBooster(multiplier, now, end, activatedBy);
    activeBooster = booster;
}
```

**Features:**
- **Thread-safe:** Uses `volatile` for the active booster reference
- **Auto-expiration:** Checks expiration on every access via `getActiveBooster()`
- **Configurable duration:** Default duration from `BoosterSettings`
- **Tracking:** Records who activated the booster

**API Methods:**
- `activateBooster(double multiplier, String activatedBy)` - Activate with default duration
- `deactivateBooster()` - Manually deactivate the current booster
- `getCurrentMultiplier()` - Get active multiplier or 1.0 if none
- `getRemainingSeconds()` - Time remaining before expiration
- `checkExpiration()` - Manually trigger expiration check

### PersonalBoosterManager

**Location:** `boosters/managers/PersonalBoosterManager.java:9`

Manages personal boosters for all players using a concurrent map.

```java
private static final Map<UUID, List<PersonalBooster>> playerBoosters 
    = new ConcurrentHashMap<>();

public static void activateBooster(UUID playerId, int index) {
    List<PersonalBooster> boosters = playerBoosters.get(playerId);
    if (boosters != null && index >= 0 && index < boosters.size()) {
        PersonalBooster booster = boosters.get(index);
        booster.activate(BoosterSettings.getPersonalBoosterDuration());
    }
}
```

**Features:**
- **Multiple boosters per player:** Players can store multiple inactive boosters
- **One active at a time:** Only one booster can be active per player
- **Storage expiration:** Unused boosters expire after configured days
- **Level-based multipliers:** 5 levels with increasing bonuses

## TP Integration

### BoosterTPAPI

**Location:** `boosters/integration/BoosterTPAPI.java:9`

Provides the bridge between boosters and Dragon Block C's TP system.

```java
public static void giveTPsWithBooster(Player player, int baseTPs) {
    IDBCPlayer dbcPlayer = NpcAPI.Instance().getPlayer(player.getName()).getDBCPlayer();
    
    // Calculate combined multiplier
    double globalMultiplier = GlobalBoosterManager.getCurrentMultiplier();
    double personalMultiplier = PersonalBoosterManager.getActiveMultiplier(player.getUniqueId());
    double combinedMultiplier = globalMultiplier * personalMultiplier;
    
    // Apply and award TPs
    int totalTPs = (int) Math.round(baseTPs * combinedMultiplier);
    dbcPlayer.setTP(dbcPlayer.getTP() + totalTPs);
}
```

<Note>
**Multiplier Stacking:** Global and personal boosters multiply together. For example:
- Base TPs: 100
- Global booster: 2.0x (100%)
- Personal booster: 1.5x (50%)
- **Final TPs: 300** (100 × 2.0 × 1.5)
</Note>

**API Methods:**

| Method | Description | Returns |
|--------|-------------|--------|
| `giveTPsWithBooster(Player, int)` | Award TPs with active boosters | void |
| `calculateBoostedTPs(Player, int)` | Calculate TPs without awarding | int |
| `getCombinedMultiplier(Player)` | Get player's total multiplier | double |
| `hasActiveBooster(Player)` | Check if any booster is active | boolean |
| `getBonusTPs(Player, int)` | Calculate bonus TPs from base | int |

### Usage Example

```java
// In your custom quest or mob kill handler
import org.debentialc.boosters.integration.BoosterTPAPI;

@EventHandler
public void onMobKill(EntityDeathEvent event) {
    Player player = event.getEntity().getKiller();
    if (player == null) return;
    
    int baseTPs = 100; // Base TP reward
    
    // Award TPs with booster multipliers applied
    BoosterTPAPI.giveTPsWithBooster(player, baseTPs);
    
    // Get the actual amount awarded
    int totalTPs = BoosterTPAPI.calculateBoostedTPs(player, baseTPs);
    player.sendMessage("§aAwarded: " + totalTPs + " TPs");
}
```

└── personal/                            # Player-specific boosters
    ├── <uuid>/
    │   └── boosters.json               # Player's booster inventory
    └── ...

**Key Operations:**

```java
// Save global booster
public static void saveGlobalBooster(GlobalBooster booster) {
    Path filePath = Paths.get(BASE_PATH, GLOBAL_FILE);
    String json = GSON.toJson(booster);
    Files.write(filePath, json.getBytes(StandardCharsets.UTF_8));
}

// Load on startup
public static void loadAllData() {
    GlobalBooster global = loadGlobalBooster();
    if (global != null && !global.hasExpired()) {
        GlobalBoosterManager.restoreBooster(global);
    }
}
```

<Warning>
Boosters are automatically saved:
- On server shutdown via `BoosterModule.shutdown()`
- When manually activated/deactivated
- Periodically by the task scheduler

Manual save calls are rarely needed.
</Warning>

## Data Models

### GlobalBooster Model

**Location:** `boosters/models/GlobalBooster.java:6`

```java
public class GlobalBooster implements Serializable {
    private double multiplier;           // e.g., 2.0 for 100% bonus
    private Instant startTime;           // When activated
    private Instant endTime;             // When it expires
    private boolean active;              // Current state
    private String activatedBy;          // Admin who activated
    
    public long getRemainingSeconds() {
        long remaining = endTime.getEpochSecond() - Instant.now().getEpochSecond();
        return Math.max(0, remaining);
    }
    
    public boolean hasExpired() {
        return Instant.now().isAfter(endTime);
    }
}
```

### PersonalBooster Model

**Location:** `boosters/models/PersonalBooster.java:8`

public class PersonalBooster implements Serializable {
    private UUID playerId;               // Owner
    private int level;                   // 1-5 (Minor to Ultimate)
    private double multiplier;           // Calculated from level
    private Instant activationTime;      // When player activated it
    private boolean active;              // Currently active flag

    public String getLevelName() {
        switch (level) {
            case 1: return "Minor";
            case 2: return "Standard";
            case 3: return "Powerful";
            case 4: return "Extreme";
            case 5: return "Ultimate";
            default: return "Unknown";
        }
    }

## Troubleshooting

<Accordion title="Booster not applying to TPs">
  **Symptoms:** Players receive base TPs without multiplier
  
  **Solutions:**
  1. Verify booster is active: `/booster info`
  2. Check if using `BoosterTPAPI.giveTPsWithBooster()` instead of direct DBC calls
  3. Ensure PlaceholderAPI is installed if using placeholders
  4. Check console for initialization errors
</Accordion>

<Accordion title="Booster disappeared after restart">
  **Symptoms:** Active booster lost on server restart
  
  **Solutions:**
  1. Check `plugins/Debentialc/data/boosters/global.json` exists
  2. Verify no errors in console during `loadAllData()`
  3. Check file permissions on the data directory
  4. Look for JSON parsing errors in logs
</Accordion>

<Accordion title="Placeholders showing empty values">
  **Symptoms:** Placeholder displays blank or not found
  
  **Solutions:**
  1. Ensure PlaceholderAPI is installed and loaded
  2. Verify placeholder registration in console logs
  3. Check placeholder syntax: `%boosters_<type>%`
  4. Try `/papi parse me %boosters_global_active%`
</Accordion>

## Advanced Usage

### Custom Booster Events

You can hook into the booster system for custom events:

```java
import org.debentialc.boosters.managers.GlobalBoosterManager;
import org.bukkit.Bukkit;

// Monitor when global boosters activate
Bukkit.getScheduler().runTaskTimer(plugin, () -> {
    if (GlobalBoosterManager.isBoosterActive()) {
        double multiplier = GlobalBoosterManager.getCurrentMultiplier();
        long remaining = GlobalBoosterManager.getRemainingSeconds();
        
        // Broadcast to all players
        if (remaining % 600 == 0) { // Every 10 minutes
            Bukkit.broadcastMessage(
                String.format("§6Global Booster: §e%.2fx §7(§f%d minutes remaining§7)",
                    multiplier, remaining / 60)
            );
        }
    }
}, 20L, 20L);
```

### Dynamic Rank Multipliers

```java
import org.debentialc.boosters.core.BoosterSettings;
import org.bukkit.entity.Player;

public double getPlayerRankMultiplier(Player player) {
    // Check player's permission groups
    for (String rank : Arrays.asList("hakaishin", "kaio", "kami", "vip", "elite")) {
        if (player.hasPermission("rank." + rank)) {
            return BoosterSettings.getRankMultiplier(rank);
        }
    }
    return 1.0; // Default: no bonus
}
```
